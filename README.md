## (A) feladat:
A ReadySetExam egy vizsgákra való felkészülést segítő webalkalmazás, amely rendszerezett formában teszi lehetővé a tantárgyak és a hozzájuk kapcsolódó feladatok hatékony kezelését.
A platformon a hallgatók interaktív módon követhetik nyomon tanulási folyamatukat, megjelölve a már elsajátított vagy további ismétlést igénylő tananyagokat.
Technikai hátterét egy React frontend és egy biztonságos, aszinkron JSON-kommunikációt használó PHP-MySQL backend biztosítja a zökkenőmentes felhasználói élmény érdekében.

## (B) feladat:
A beadandó elméleti része a JSON-formátumú adatkezelést és az aszinkron kommunikáció (AJAX) szerepét mutatta be a modern webfejlesztésben.
A dolgozat részletezte, hogyan váltja fel a JSON a nehézkesebb XML-t, biztosítva a gyors és hatékony adatcserét a React kliens és a PHP szerver között oldalújratöltés nélkül.
A leírás kitér a biztonsági szempontokra is, különös tekintettel a szerveroldali PDO-alapú adatkezelésre, szemléltetve az alkalmazásban használt technológiai megoldások gyakorlati előnyeit.
